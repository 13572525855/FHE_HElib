实现：
  用java编写Jcli_FHE和Jser_FHE程序，在其中调用共享库，声明本地方法service()和client()（未实现）并使用。用Qt使用C++编写java程序中对应方法的具体实现，并编译成共享库文件。将共享库放到jdk的lib中，并使共享库路径指向lib。实现了java使用JNI技术调用同态加密客户端程序和服务器端程序，即实现了对同态加密各函数的java调用。


存在问题：

	2.由于程序是基于全同态加密库的，编译共享库的过程对库的依赖性和紧密性很强。

其他文件说明：
	1.Jcli_FHE.h是Jcli_FHE.java编译成的头文件，里面提供用C++实现具体方法时的函数名。
